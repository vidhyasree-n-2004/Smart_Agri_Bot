# 🌱 Smart Agri Bot

## 📌 Overview
Smart Agri Bot is an AI and IoT-based agricultural robot designed to assist farmers in monitoring crop conditions and automating irrigation. The system collects real-time environmental data using sensors and helps improve farming efficiency while reducing water wastage and manual effort.

## 🚀 Features
- Soil Moisture Monitoring
- Temperature and Humidity Monitoring
- Automatic Irrigation Control
- Real-Time Data Collection
- Obstacle Detection
- Smart Farming Assistance
- User-Friendly Monitoring Interface

## 🛠️ Technologies Used
### Hardware
- Arduino / NodeMCU
- Soil Moisture Sensor
- DHT11/DHT22 Sensor
- Water Pump
- Relay Module
- Ultrasonic Sensor
- Power Supply

### Software
- Embedded C
- Arduino IDE
- IoT Integration
- HTML, CSS, JavaScript (for dashboard, if applicable)

## ⚙️ Working Principle
1. Sensors collect soil moisture, temperature, and humidity data.
2. The microcontroller processes the sensor readings.
3. If soil moisture falls below a predefined threshold, the irrigation system is activated automatically.
4. Environmental data can be displayed on a monitoring dashboard.
5. The robot assists in efficient crop management and water conservation.

## 🎯 Objectives
- Reduce manual farming effort.
- Improve water management.
- Enable smart and sustainable agriculture.
- Increase crop productivity through automation.
